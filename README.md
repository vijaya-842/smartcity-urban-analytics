# SmartCity Twin — Urban Analytics with PySpark

> Big Data Analytics Project — Apache Spark (PySpark) · NYC Open Data (2025)

## Overview
A scalable urban analytics platform that processes 5M+ NYC open dataset records using Apache Spark (PySpark), performing distributed data cleaning, aggregation, and trend analysis to surface actionable insights on traffic, service requests, and city infrastructure.

## Key Features
- **Large-Scale Data Processing** — Distributed computation over 5M+ NYC open dataset records
- **Urban Intelligence** — Insights on traffic patterns, service requests, and infrastructure across NYC boroughs
- **Feature Engineering** — Statistical analysis pipelines for extracting actionable urban signals
- **Interactive Visualizations** — Dashboards and reports for both technical and non-technical stakeholders

## Tech Stack
| Category | Tools |
|---|---|
| Big Data Processing | Apache Spark (PySpark) |
| Data Source | NYC Open Data |
| Analysis | Python, Pandas, NumPy |
| Visualization | Matplotlib, Seaborn |

## Datasets Used
- NYC 311 Service Requests
- NYC Traffic Volume Counts
- NYC Infrastructure & Borough Data

## Analysis Pipeline
```
NYC Open Data (5M+ records)
        ↓
PySpark Distributed Cleaning & Aggregation
        ↓
Feature Extraction & Statistical Analysis
        ↓
Urban Insights → Interactive Visualizations & Report
```

## Key Insights
- Traffic pattern trends across NYC boroughs
- Service request hotspots and resolution times
- Infrastructure usage patterns and anomalies
